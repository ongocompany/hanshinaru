```javascript
const [authorsDB, poemsCompact, poemsFull, historyCards] = await Promise.all([
```

#### Step 3: 타임라인 렌더링 (app.js:741-755)

```javascript
// 1) 연도별 그룹화
const aByYear = groupByYear(authorEvents);  // 시인 이벤트
const hByYear = groupByYear(historyEvents); // 역사 이벤트
const years = [모든 연도 합치고 정렬];

// 2) 연도별 렌더링
for (const y of years) {
  const left = (aByYear.get(y) || []).slice(0, 2).map(renderAuthorCard);  // 좌측 최대 2개
  const right = (hByYear.get(y) || []).map(renderHistoryCard);            // 우측 전체
  root.appendChild(renderPrimaryItem(y, left, right));
}
```

**레이아웃 구조**:
```

├─ [timeline-left]   ← 시인 카드 (최대 2개)
├─ [timeline-marker] ← 연도 표시
└─ [timeline-right]  ← 역사 카드 (전체)
```

## 🖱️ 4. 이벤트 흐름

### 4-1. 아코디언 (카드 펼치기)

**바인딩**: `bindAccordions()` (app.js:216-242)

```
사용자 클릭 (.works-toggle)
    ↓
이벤트 위임 (root.addEventListener)
    ↓
현재 카드 expanded 상태 확인
    ↓
다른 카드 닫기 (querySelectorAll(".card.expanded"))
    ↓
현재 카드 토글
    ↓
aria-expanded 업데이트
```

**특징**:
- 한 번에 하나의 카드만 열림
- `.works-panel` hidden 속성으로 제어
- aria-expanded로 접근성 확보

---

### 4-2. 모달 열기

**바인딩**: `bindModalOpeners()` (app.js:575-634)

#### 경로 1: 작품 클릭 → 시인 모달

```
.work-item[data-poem-id] 클릭
    ↓

    ↓
poet.zh로 authorId 매칭
    ↓

    ↓
시인 모달 열림 + 해당 작품 자동 펼침
```

```
    ↓

    ↓
시인 모달 열림
```

**코드 위치**: app.js:615-623

#### 경로 3: 역사 카드 클릭 → 역사 모달

### 8-2. fetch 구조 유지 가능 여부

**✅ 가능**

```javascript
// 기존 (app.js:722-727)
const [authorsDB, poemsCompact, poemsFull, historyCards] = await Promise.all([
  loadJSON("public/index/db_author.with_ko.json"),
  loadJSON("public/index/poems.compact.json"),
  loadJSON("public/index/poems.full.json").catch(() => poemsCompact),
  loadJSON("public/index/history_cards.json"),
]);

// 백엔드 연결 후 (변경 최소화)
const [authorsDB, poemsCompact, poemsFull, historyCards] = await Promise.all([
  loadJSON("/api/authors"),              // ← URL만 변경
  loadJSON("/api/poems?type=compact"),   // ← URL만 변경
  loadJSON("/api/poems?type=full").catch(() => poemsCompact),
  loadJSON("/api/history"),              // ← URL만 변경
]);
```

**조건**:
- API 응답 스키마가 현재 JSON 파일과 동일
- `loadJSON()` 함수 그대로 사용 가능 (fetch 기반)
