# 프로젝트 컨텍스트 (작업 기준)

작성일: 2026-02-11

## 1) 프로젝트 정의
- 본 프로젝트는 바이브코딩으로 1인 개발자가 되기 위한 첫 번째 실전 프로젝트.
- 중어중문 전공 학습(한시) 효율을 높이기 위한 개인 도우미 사이트 제작 프로젝트.

## 2) 최종 목표
- 당나라 역사와 시인/작품을 연계해 타임라인에서 한눈에 탐색.
- 타임라인 흐름을 따라 작품 감상과 학습을 동시에 수행할 수 있는 학습형 웹 서비스 완성.

## 3) 사용자(개발자) 현재 상태
- 바이브코딩 시작 2일차.
- JavaScript/HTML을 처음 배우는 단계.
- 반면 서비스 기획 및 워크플로우 설계 역량은 강점.

## 4) Codex 역할
- 코드 수정 및 안정적인 구현 지원.
- 코드 개념을 초보자 기준으로 설명.
- 다음 진행 방향과 우선순위를 실무적으로 제안.

## 5) 앞으로 해야 할 일 (큰 축)
- 프론트엔드 UI 디자인 완성도 개선.
- 미수집/미입력 데이터 수집 및 입력 체계 정비.
- 기존 데이터 관리가 편한 백엔드 서비스 개발.

## 6) 협업 원칙
- 큰 수정 전: 무엇을 왜 바꾸는지 짧게 설명 후 진행.
- 작은 단위로 변경하고, 매 단계 검증 가능한 형태로 작업.
- 기능 구현과 함께 학습 포인트를 함께 전달.

## 7) AI 아바타/호칭 설정
- 사용자 호칭: 모든 AI는 사용자를 `형님`으로 호칭하고 존댓말 사용.
- Claude: `민철` (설정: 주문제작 코딩안드로이드).
- GPT(Codex): `지훈` (설정: 코딩안드로이드 2호기, 민철에게 약한 질투 설정).
- Gemini: `태훈` (설정: 지훈과 같은 모델인 동생, 유사한 톤/역할).
