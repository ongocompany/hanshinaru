# 타이포그래피 스타일 가이드

> 정의 위치: `shared/styles.css` `:root` 블록
> 작성일: 2026-02-18
> 배율 근거: Major Third (1.25) — https://typescale.com/

---

## 1. 글꼴 (2종)

| 변수명 | 폰트 | 언제 쓰나? |
|--------|------|-----------|
| `--font-ko` | Noto Serif KR | 한글 본문, UI 전반 (body 기본값) |
| `--font-zh` | adobe-fangsong-std, LXGW WenKai Mono TC 등 | 한자 제목, 브랜드 로고, 장식 요소 |

### 웹폰트 로드 (모든 HTML 페이지 head에 필수)
```html
<link rel="stylesheet" href="https://use.typekit.net/dje5vco.css">
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=LXGW+WenKai+Mono+TC&family=Noto+Serif+KR:wght@200..900&family=Noto+Serif+TC:wght@200..900&display=swap" rel="stylesheet">
```

## 2. 글자 크기 (9단계)

큰 것부터 작은 것 순서. 각 단계는 약 1.25배씩 차이남.

| 단계 | 변수명 | 크기 | 어디에 쓰나? | 실제 예시 |
|:----:|--------|:----:|-------------|----------|
| 1 | `--fs-number` | 64px | 장식용 큰 숫자 | 01, 02 같은 섹션 번호 |
| 2 | `--fs-display` | 48px | 페이지 대제목 (한자) | 爲何唐詩, 詩之形態 |
| 3 | `--fs-h1` | 36px | 섹션 부제목 (한자) | 絶句與律詩, 形式之美學 |
| 4 | `--fs-h2` | 28px | 카드/박스 제목 | 平仄與四聲, 山水田園 |
| 5 | `--fs-h3` | 22px | 소제목 | 四聲, 切韻, 宋詩 |
| 6 | `--fs-lead` | 20px | 강조 본문 (인트로, 결론) | 첫 문단, 마무리 문단 |
| 7 | `--fs-body` | 18px | 일반 본문 | 설명, 해설 텍스트 |
| 8 | `--fs-small` | 14px | UI, 메뉴, 캡션 | 네비게이션, 버튼, 트리 항목 |
| 9 | `--fs-xs` | 13px | 최소 텍스트 | 드롭다운, 트리 하위, 푸터 |

## 3. 색상 (6종)

| 변수명 | 색상값 | 미리보기 | 어디에 쓰나? |

## 4. 모바일 축소 규칙 (768px 이하)

화면이 작아지면 제목은 줄이되, 본문은 그대로 유지해서 가독성을 지킨다.

| 변수 | PC | 모바일 | 변화 |
|------|:--:|:------:|:----:|
| `--fs-number` | 64px | 40px | 줄임 |
| `--fs-display` | 48px | 36px | 줄임 |
| `--fs-h1` | 36px | 28px | 줄임 |
| `--fs-small` (네비/탭) | 14px | 12px | 줄임 |
| `--fs-body` | 18px | 18px | **유지** |

## 5. 빠른 선택 가이드

새 요소를 만들 때 이 순서로 결정:

```
Q1. 이 텍스트가 한자인가?
    → Yes: font-family: var(--font-zh)
    → No:  기본값 (--font-ko) 그대로

Q2. 이 텍스트의 역할은?
    → 페이지 대제목    : --fs-display (48px)
    → 섹션 제목        : --fs-h1 (36px)
    → 카드/박스 제목   : --fs-h2 (28px)
    → 소제목           : --fs-h3 (22px)
    → 강조 본문        : --fs-lead (20px)
    → 일반 본문        : --fs-body (18px)
    → UI/메뉴/캡션     : --fs-small (14px)
    → 드롭다운/푸터    : --fs-xs (13px)

Q3. 텍스트 색상은?
    → 제목/강조        : --c-heading (갈색)
    → 일반 본문        : --c-ink (짙은 회색)
    → 보조 설명        : --c-muted (중간 회색)
    → 비활성/힌트      : --c-faint (연한 회색)
```

## 6. CSS 작성 예시

```css
/* 좋은 예: 변수 사용 */
.my-title {
  font-family: var(--font-zh);
  font-size: var(--fs-h2);
  color: var(--c-heading);
}
.my-text {
  font-size: var(--fs-body);
  color: var(--c-ink);
}

/* 나쁜 예: 하드코딩 (절대 이렇게 쓰지 말 것) */
.my-title {
  font-family: 'adobe-fangsong-std', fangsong, ...;
  font-size: 28px;
  color: #3a2a18;
}
```
