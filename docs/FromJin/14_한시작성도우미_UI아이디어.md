## **1단계: 영감 포착 (立意, 입의)**

먼저 무엇을 쓸지 정해야 합니다. 초보자에게는 멀리 있는 추상적인 관념보다 **지금 눈앞에 보이는 풍경**을 추천합니다.

- **상황:** 퇴근길, 지친 몸을 이끌고 집에 가는데 서쪽 하늘에 붉은 노을이 걸려 있고, 문득 가족이 보고 싶어집니다. (여기까지는 사용자가 작성)
- **핵심 키워드:** 노을(晩霞/落日), 지친 몸(倦軀), 집(家/歸), 그리움(思). (핵심 키워드는 AI가 추천, 혹은 직접 선정)
- **코칭 Tip:** 너무 많은 것을 담으려 하지 마세요. '노을을 보며 집에 가고 싶다'는 하나의 감정에만 집중하십시오. (AI, 혹은 고정된 팁)

## **2단계: 시상 전개 (起承轉結, 기승전결) 구성 (시체 선택)**

- 2단계 들어가기전 시체 선택 (임시저장) 아래 부분 해 보고 절구가 아니라 율시로 가야 한다고 판단하면 다시 진행

작업에 들어가기 전, 4줄의 역할을 나눕니다.  (네줄의 입력칸 생성)

1. **기(起):** 시상을 불러일으킴 (풍경 묘사) -> *해가 지고 노을이 번진다.*
2. **승(承):** 시상을 이어받아 심화함 -> *하루의 일과를 마치고 돌아가는 길.*
3. **전(轉):** 시상을 전환하여 감정을 고조시킴 -> *문득 고개를 드니 밝은 달이 뜬다.* (혹은 다른 생각으로 전환)
4. **결(結):** 여운을 남기며 마무리 -> *내 집의 등불이 그리워지네.*
- 앞의 설명은 AI → 뒤 부분은 인간이 작성
- 정리되고 마음에 들면 다음 단계로  → 시체가 결정되었으니 화면에 시체에 맞는 UI 표시 (원고지같은 격자무늬 사각 테이블, 한지 느낌 바탕, 먹선 같은 줄느낌, 각 글자 입력칸 하단에 훈음,평측,신호등-원칙에 따라 맞는지 아닌지 보여주는 표시, 이렇게 표시지가 들어가게 구성. 일단 편의상 이 UI를 <원고지>라고 부르기로함

## **3단계: 압운(押韻)과 운자 선택**

한시의 핵심 규칙인 '운(Rhyme)'을 먼저 정해야 합니다. 오언절구는 보통 **2구와 4구의 끝자**에 같은 운부의 글자를 놓습니다. (1구 끝은 선택사항) → 기식 차이(평기식, 측기식)도 여기서 결정, 수구 압운 여부도 여기서 결정 (측기식 수구불압운이 정격이나 그렇지 않다고 해도 금지는 아니므로 강제하지 않음. 다만 초보자들에게 설명은 해줘야 함)

1. 압운방식에 따라 압운자 선택 (수구압운 불압운에 따라서 위치 변동)
2. **운부 선택:** 이때 운목을 통해 압운자 선택하는 UI : 원고지 하단에, 먼저 운목 106자 테이블을 보여줌 (이때 운목 글자 하단에 훈음을 보여줄지 아니면 마우스오버로 할지 고민) - 원하는 운목 클릭하면 운목에 속하는 한자들 나열 (숫자가 많을수도 있으니 여기서 어떻게 보여줄지가 문제..각각 하단으로 아코디언 형태로 보여주는게 좋을듯. 운목고를때 먼저 한번 펼치고 고르고나면 운목은 닫으면서 소속 한자들 나열하는 식으로 펼치고) 마음에 드는 운자를 고르고 닫기를 누르면 아코디언 닫히면서 원고지에 해당 글자 자동 입력
    1. 자신이 작성한 한글 초고를 보고 나머지 운자를 모두 고를지,  
    2. 아니면, 시상을 전개하며 고를지 결정
3. 2-a 이제 나머지 한글자 혹은 두글자 (수구 압운이냐 아니냐에 따라) 운자를 고르면 되는데 운목은 이미 정해졌으니 운목에 해당하는 한자들만 보여줌 (2번 반복)
4. 2-b 의 경우 바로 4단계로

1. 3 단계에서 2-a를 선택한 경우 
    1. 각종 원칙 (고평금지, 하삼련금지 등등 원칙과 측기식으로 갈것인지 평기식으로 갈것인지 등등)에 따라 해당 자리에 각각 적당한 한자를 찾아 넣음
    2. 실시간으로 형식에 맞는 글자인지 판단해 글자 하단에 표시 (빨간불, 파란불, 노란불 등)
    3. 다 완성해서 적어 넣은 후 5단계
2. 3단계에서 2-b를 선택한 경우
    1. 일단 자신이 2 단계에서 완성한 시상전개에 따라 적당한 한자를 각 칸에 다 넣음

## **5단계: 전문가(AI)의 퇴고 (평측 및 격률 검사)**

1. 먼저 자신이 쓴 한시의 번역을 보여줌 (AI) 이게 중요한게 한국 사람이기 때문에 한자로만 문장을 만들어 놓으면 자신이 의도한 뜻과 다르게 될 수도 있음. 그게 시의 매력이긴 하지만 일단 의미 전달이 되는가 안되는가의 판단은 필요 (이거는 3-2-a, 3-2-b 모두 동일)
2. 그 다음 평측 및 격률 각종 병 표시  (3-2-a 라면 앞단계에서 이미 다 끝났을 것이므로 안보여줘도 됨)
3. 원고지 수정 
4. 다시 검수 ((격식은 맞는데 의미가 이상하거나, 격식도 맞고 의미도 맞는데 대장이 안됐거나-절구는 대장이 필수는 아니지만 하면 좋으니. 이런 거를 AI가 검수때마다 판단하여 도와줌. 회원 레벨에 따라 검수 횟수 제한)
5. 최종 완성
    1. 완성된 5언 절구를 화면에 서예폰트로 띄워줌
    2. 개인 시첩에 저장, 공유, 혹은 시회만들기로 넘어감
    3. 시화 만들기를 선택하면 시에 어울리는 배경화면 (프리셋으로 할지, 이것도 인공지능으로 실시간 합성할지 고민-회원 레벨에 따라 다르게?)을 선택 혹은 제작하고 시를 얹어서 레이아웃을 보고 결정
    4. 저장, 공유 등 선택
