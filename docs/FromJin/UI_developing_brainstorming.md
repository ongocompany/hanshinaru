
# 시인 모달 및 시인 관리툴 UI 개선 방향

1. 현재 문제점 : 시 보기 화면이 밋밋해서 그림을 넣어보려고 하다가 전체 UI가 이상해졌음 
    1. 세로방향으로 길쭉한 시와 가로방향으로 길쭉한 설명문서들 (집평, 주석등)이 중간중간에 혼재해 있어 깔끔한 느낌이 전혀 없음
    2. 공간 절약을 하려고 민철이에게 레이아웃 변경 지시를 이것저것 하다가 더 이상해졌음. 젠장
    3. 그렇다고 그림을 포기하고 싶지는 않음. 미드저니 그림 출력 품질이 좋기도 하고 과금도 했는데 돈 아까움
    4. 우리 사이트 전체적인 디자인 (담백함. 학습, 학술 웹페이지 느낌)에서 적어도 시 본문 만큼은 한 조각 예술적인 느낌을 주고 싶은데 레이아웃 아이디어가 떠오르지 않음
2. 개선방향 고민
    1. 시/시인 모달 폭을 원래대로 900px 정도로 되돌리면 가로세로 레이아웃에서 오는 어색함이 줄어들까
    2. 그림과 시를 한 화면에 배치하는 가장 좋은 방법이 무엇일까?
    3. 그림을 크롭해서 배경과 자연스럽게 어우러지게 하고 우측 하단에 넣으면 좋을텐데, 미드저니 그림이 사각으로만 우측 하단에 넣으면 어색함
    4. 게다가 동양화는 원래가 여백에 시를 써 넣는 전통이 있다보니 그림이 바닥에 깔리고 그 위에 시가 올라가면 기가막히게 멋짐 —> 욕심을 못 버리고 있는 이유…
    5. 시가 매우 길어지는 경우도 문제임.. 절구나 율시는 길이가 정해져 있어서 괜찮은데, 고시나 악부시 중에 미치게 긴 시들 (특히 백거이 시들)은 어찌 처리하지?
3. AI 동생들에게 요청사항
    1. 나의 위 고민들에 대해서 관련 파일들 검토해 보고 현재 문제점. 개선 방향 아이디어 제안 부탁함 (나 진지함.. 스트레스 받는 중)
    2. 우리사이트와 유사한 사이트중 좋은 디자인으로 평가 받는 사이트가 있으면 검색해서 추천 바람 (언어 상관 없음)
