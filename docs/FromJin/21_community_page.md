커뮤니티 페이지 구성안
전체 페이지 2컬럼 레이아웃 : 왼쪽 메뉴, 오른쪽 본문
이후 하단 본문페이지 설명
상단 2컬럼 그리드 (컬럼폭 500px, 높이 400px) 
좌측: 공지사항 (3줄만 노출, 하단은 광고)
우측: 자유게시판 (기존자유게시판) 기존 필드에서 사이즈에 맞게 노출될 항목만 선정. 작성자, 제목, 날짜, 조회수, 추천수
가는줄 (1000px, 1px )
하단 2컬럼 그리드 
좌측: 뉴스 영역 (가로 500px, 세로 400px) 플레이스 홀더
우측: 묻고 답하기 (가로 500px, 세로 400px): 신설, 회원들이 묻고 답하는 공간, 자유게시판 폼 그대로 사용 
중간 광고 영역 (가로 1000px, 세로 100px) 플레이스 홀더
가는줄 (1000px, 1px )
창작마당 (기존한시자랑) : 1컬럼 (1000px, 400px) 신규 폼 개발 필요 (시 입력하고, 한문서체 고르고, 배경화면 까지 골라서 에디토리얼 형태로 한장의 족자처럼 완성해주는페이지) 창작마당에 있는 시중에서 추천수 높은것 4개만 노출(노출개수는 썸네일형태에따라 변경)

이후 왼쪽 사이드 메뉴 설명
좌측 사이드 메뉴에 상단 각 영역 제목 노출 (공지사항, 자유게시판, 뉴스, 묻고답하기, 창작마당)
각 영역 클릭하면 해당 페이지로 이동

--- 정리 ---

공지사항 페이지 : 기존 공지사항 페이지 그대로 사용 (자유게시판 입력폼 사용)
자유게시판 페이지 : 기존 자유게시판 페이지 그대로 사용
뉴스 페이지 : 신설, AI가 작성한 한시관련 뉴스를 보여주는 페이지, 메인페이지와 동일한 내용
묻고 답하기 페이지 : 신설, 회원들이 묻고 답하는 공간, 자유게시판 폼 그대로 사용
창작마당 페이지 : 기존한시자랑 페이지 그대로 사용 (신규입력폼 및 게시판 형태 개발)
