# UI 관리툴작업지시

1.  관리툴 위치 : 데이터 관리툴 데이터 검증 탭 옆
2. 워크 플로우
    1. 오픈과 동시에 데이터 로딩
    2. 상단 서브 탭에서 시대 선택 : 하단에 현재 UI 세팅 보여주고 수정 가능하게 표시 (디폴트는 현재값)
        1. 시대별 타임라인(메인페이지) 배경색 및 글자색
        2. 시인(작품모달) 섹션별 배경색 및 글자색 (시인 부분은 필요 없고, 하단 작품쪽 섹션별 색상 선택 가능하게…)
            1. 작품리스트 배경색 (작품모달 하단에 붙는 리스트)
            2. 시 본문
            3. 집평(한자)
            4. 집평(한글)
            5. 심화자료
            6. 멀티미디어자료
        3. 추가 확인  
            1. 현재 캐릭터 세트 어느 부분에 어느 폰트가 쓰이는지 리스트로 확인
            2. 각각 세트별로 수정가능한지 (예를 들면 제목 - noto - 20pixel - 컬러넘버) 이런식으로.. 되어 있을때 저게 어디어디에 쓰이고 있는지 리스트로 설명해주고, 변경시 어디가 바뀌는지 알고 싶음
            3. 캐릭터 셋 수정 가능하다면 수정할 수있게 해줘 (폰트, 크기, 색깔)
    3. 미리보기 버튼 및 취소 버튼
    4. 저장 (저장이 CSS만 되는지 Json도 건드려야되는지 확인)
3. UI 관리툴의 UI 관련 추가 확인 사항
    1. 상단에서 시대별로 탭 나누지 말고 4컬럼으로 만들어서 한눈에 보면서 수정도 가능할지
    2. 아니면 전체보기 / 시대별 보기로 나눠서 작업 가능한지
