# 질문

- board seed `西樓`를 정지상의 별도 작품으로 계속 추적해야 하는지, 아니면 이미 확보한 `栢律寺` 계열 원문과 같은 작품의 이칭으로 볼 수 있는지 점검한다.

# 확인한 근거

- 한국민족문화대백과 정지상 항목은 정지상의 작품이 `『동경잡기(東京雜記)』에 「백률사(栢律寺)」·「서루(西樓)」등이 전한다`고 적고 있다.
  - https://encykorea.aks.ac.kr/Article/E0050946
- 1차 공개 원문 `동경잡기(東京雜記)/卷二`에는 `栢栗寺` 항목 아래 `全思敬西樓記`가 이어지고, 그 바로 뒤에 정지상 시 전문이 실려 있다.
  - 해당 페이지에서는 정지상 시의 표제가 `西樓`로 따로 분리되어 보이지 않고, `栢栗寺` 섹션 안에서 tower/樓 맥락과 함께 전한다.
  - https://zh.wikisource.org/zh-hant/東京雜記/卷二
- 병행 원문 `新增東國輿地勝覽/卷021`도 같은 시를 `柏栗寺` 항목 아래 전한다.
  - https://zh.wikisource.org/wiki/新增東國輿地勝覽/卷021
- 2차 연구자료 중 성신여대 박사논문 `茶事典籍을 通해 본 宋과 高麗의 茶文化 考察`의 표 7은 정지상의 작품명을 `백률사서루(栢栗寺西樓)`로 적고 있다.
  - 즉 후대 연구/정리에서는 이 시를 `栢栗寺` 단독 표제 대신 `栢栗寺西樓`로 묶어 부르는 사례가 확인된다.
  - https://repository.sungshin.ac.kr/bitstream/2025.oak/2839/2/%E8%8C%B6%E4%BA%8B%E5%85%B8%E7%B1%8D%EC%9D%84%20%E9%80%9A%ED%95%B4%20%EB%B3%B8%20%E5%AE%8B%EA%B3%BC%20%E9%AB%98%E9%BA%97%EC%9D%98%20%EF%A7%BE%E6%96%87%E5%8C%96%20%E8%80%83%E5%AF%9F.pdf

# 현재 판단

- `西樓`가 완전히 별개의 미수집 작품이라고 단정하기는 어렵다.
- 현재 공개 1차 원문에서는 정지상 시가 `栢栗寺/柏栗寺` 항목 아래 직접 열리고, `西樓`는 tower context 또는 prose heading(`西樓記`)로만 확인된다.
- 반면 2차 자료에서는 이 시를 `栢栗寺西樓`로 호명하므로, board seed `西樓`는 이미 확보한 `栢律寺` 계열 시의 약칭/이칭일 가능성이 높다.

# 운영 결론

- 당장은 `西樓`를 exact-title direct-text 확보 완료로 닫지 않는다.
- 대신 `西樓 ↔ 栢栗寺西樓 ↔ 栢律寺/柏栗寺 수록시`의 이칭 가능성을 연구 메모로 고정한다.
- 이후 primary source에서 `西樓` 또는 `栢栗寺西樓`가 명시된 heading/목차/문집 표제를 찾으면 그때 seed 해소로 승격한다.
