# 창작마당 입력폼 강화 — 배경 이미지 + 폰트 선택 개발계획서

> **작성자**: 민철 (Claude)
> **작성일**: 2026-02-22
> **프로젝트**: tangshi.kr — 창작마당(Showcase)

---

## 1. 개요

창작마당 게시판에 시각적 표현력을 추가한다.

- **배경 이미지**: 템플릿 선택(무료) + AI 생성(프리미엄, NanoBanana API)
- **한자 폰트 선택**: Google Fonts 서예체 5~6종 + 실시간 미리보기
- **족자 스타일 디스플레이**: 배경+폰트+세로쓰기 통합 렌더링

---

## 2. DB 스키마 변경

`posts` 테이블에 4개 컬럼 추가:
```sql
ALTER TABLE public.posts
  ADD COLUMN bg_type text,           -- 'template' | 'ai' | null
  ADD COLUMN bg_template_id text,    -- 템플릿 파일명
  ADD COLUMN bg_image_url text,      -- AI 생성 이미지 URL
  ADD COLUMN font_style text;        -- 폰트 ID
```
