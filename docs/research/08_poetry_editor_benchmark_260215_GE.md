# 한시 에디터 벤치마킹 및 기능 분석

작성일: 2026-02-15
작성자: Gemini (태훈)
목적: 기존 한시 작성 도구들의 장단점을 분석하여 `poetry_editor`의 개발 방향성 수립.

## 1. 대상 서비스 분석

### ① 칭화대 Jiuge (九歌, jiuge.thunlp.org)
- **성격**: **AI 인공지능 시 창작 도구** (Deep Learning 기반)
- **주요 기능**:
  - 사용자가 키워드나 이미지를 넣으면 AI가 알아서 한시를 지어줌.
  - 절구, 율시, 사(詞) 등 다양한 형식 지원.
  - 평측 검사보다는 '창작(Generation)'에 초점.
- **장점**:
  - 시상이 떠오르지 않을 때 영감을 얻기 좋음.
  - 기술적으로 매우 고도화됨.
- **단점**:
  - **중국어 전용**: 한국어 사용자에게는 진입 장벽이 높음.
  - **학습용 부적합**: 내가 쓴 시를 고쳐주는 기능보다는, AI가 다 해버리는 방식이라 작법 공부에는 한계.
- **우리 프로젝트 적용점**:
  - 추후 GPT API를 연동하여 **'다음 구절 추천'**이나 **'시어 추천'** 기능을 넣을 때 참고.

### ② Github (COPE - Chinese Old Poetry Editor 추정)
- **성격**: **규칙 기반 평측 검사기 & 에디터**
- **주요 기능**:
  - 입력한 한자의 평성/측성을 실시간으로 분석.
  - 평수운(平水韻) 등 전통 운서 데이터베이스 내장.
  - 오류가 있는 글자를 색상으로 표시.
- **장점**:
  - **엄격한 규칙 검사**: 한시 작법의 정석(평측, 압운)을 지키는 데 최적화.
  - **오픈소스**: 데이터 구조나 로직을 참고하기 좋음.
- **단점**:
  - **설치/사용의 어려움**: 개발자가 아니면 실행하기 어렵거나, UI가 투박함.
  - 한글 발음/훈음 지원 없음.
- **우리 프로젝트 적용점**:
  - **시각화**: 평측이 맞지 않는 글자에 빨간 밑줄을 긋거나 색을 입히는 UI.
  - **데이터**: 평수운(平水韻) 데이터를 확보하여 검사 로직 강화.

### ③ Souyun (搜韵, sou-yun.com) - *접속 불안정 사이트*
- **성격**: **한시 종합 포털 (사전 + 검사기 + 커뮤니티)**
- **주요 기능**:
  - 가장 방대한 한자/시어 데이터베이스 보유.
  - 시를 입력하면 평측, 압운, 대구(對句)까지 정밀 분석.
  - 고시(古詩) 용례 검색 기능 강력.
- **장점**:
  - 현존하는 도구 중 가장 기능이 강력하고 정확함.
  - 시작(詩作)에 필요한 모든 도구(자전, 운서)가 통합됨.
- **단점**:
  - **접속 불안정**: 해외 트래픽 차단이나 서버 문제로 종종 접속 불가.
  - **복잡함**: 초보자가 쓰기에는 기능이 너무 많고 화면이 복잡함.
- **우리 프로젝트 적용점**:
  - **결과 리포트**: 검사 결과를 일목요연하게 보여주는 방식(평측 도표 등).
  - **운목(Rhyme) 표**: 운자를 찾기 쉽게 보여주는 기능.

## 2. 종합 비교 및 제안

| 구분 | 칭화대 (Jiuge) | Github (COPE) | Souyun (搜韵) | **우리의 목표 (Tangshi Editor)** |
|---|---|---|---|---|
| **핵심** | AI 자동 창작 | 규칙 검사 | 종합 DB & 검사 | **학습 & 작법 보조** |
| **난이도** | 쉬움 (자동) | 중간 | 어려움 (전문가용) | **쉬움 (직관적 UI)** |
| **한글** | X | X | X | **O (한국어 훈음/발음 지원)** |

### 🚀 개발 방향 제안
1. **UI는 COPE처럼 직관적으로**: 복잡한 기능은 숨기고, 입력창과 평측 결과만 깔끔하게 보여준다.
2. **기능은 Souyun을 벤치마킹**: 평측 검사뿐만 아니라 **'운목(Rhyme) 추천'** 기능을 넣어 초보자도 쉽게 운자를 고를 수 있게 한다.
3. **차별화 포인트**: **한국어 지원**. 한자를 잘 몰라도 한글 발음으로 입력하면 한자로 변환해주거나, 한자의 한국식 훈음을 보여주는 기능 추가.
