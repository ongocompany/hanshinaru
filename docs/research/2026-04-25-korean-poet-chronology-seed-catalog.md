# 한국 시인 연대기형 seed catalog v1

작성 기준: 2026-04-25 10:48 KST

## 목적

한국 한시 수집 방식을 `작품 대량 수집 우선`에서 `시인 연대기 우선`으로 바꾼다.

새 운영 방식은 다음 순서로 간다.

1. 고대부터 근대 전환기까지 시인 목록을 연대기형으로 세운다.
2. 각 시인마다 생애, 역사 배경, 문학사적 의미, 주요 저서와 작품을 먼저 정리한다.
3. 홈페이지에는 시인별 대표작만 얇게 싣지 않고, 문학사적으로 의미 있는 작품을 넉넉히 싣는다.
4. V1은 `대표작/핵심작 우선 공개`, V2는 `문집 단위 확장`, V3는 `선집·지역 문헌·OCR 보강`으로 넓힌다.

## 저작권 운영 원칙

- 원문: 저작권이 소멸한 한문 원문은 수집 가능하되, 원문 출처와 판본을 반드시 남긴다.
- 기존 번역: 위키문헌처럼 라이선스가 확인되는 번역만 사용한다.
- 한국고전종합DB 번역: 원문 확인과 참고에는 쓰되, 홈페이지 공개 번역으로 바로 복제하지 않는다.
- 의심 번역: 직접 번역과 직접 해설로 대체한다.
- 주석: 기존 주석은 출처와 권리를 확인하고, 애매하면 직접 주석을 새로 작성한다.
- 시인 bio: 백과사전식 설명은 그대로 복제하지 않고, 사실 확인 후 자체 문장으로 요약한다.

## 수집 등급

- A: 본편 V1 수집 대상. 한문 한시, 향가, 고려가요, 시조 등 갈래와 무관하게 대표작과 원문 확보 가능성이 높다.
- B: V1 또는 V2 후보. 문집/선집 접근은 가능하지만 대표작 확정 또는 권리 검토가 더 필요하다.
- C: 본편 수집 대상이지만 갈래·전승·원문 형태를 별도 처리해야 하는 대상. 향가, 고려가요, 전승 시가, 설화 속 시문을 포함한다.
- D: 보류/검증 대상. 생애·작품 귀속 또는 원문 접근이 불안정하다.

## V1/V2/V3 확장 방식

### V1: 연대기 대표 화면

- 각 시대별 A급 작가를 먼저 공개한다.
- 작가별 작품 수를 숫자로 제한하지 않고, 원문 확보와 문학사 의미가 확인된 작품을 우선 싣는다.
- 최소 목표는 작가별 `대표작 + 문집 대표 수록작 + 역사 맥락 작품` 묶음이다.

### V2: 문집 단위 확장

- `孤雲集`, `東國李相國集`, `牧隱藁`, `梅月堂集`, `退溪集`, `蘭雪軒集`, `茶山詩文集`, `梅泉集`처럼 작가별 핵심 문집을 tranche로 수집한다.
- 각 tranche는 원문, 출처 URL, 판본, 권리 상태, 자체 번역 필요 여부를 함께 기록한다.

### V3: 선집·지역 문헌·OCR 보강

- `東文選`, `三韓詩龜鑑`, `東京雜記`, 지역 읍지, 개인 문집 이미지 자료로 확장한다.
- direct-text가 없는 작품은 KORCIS 소장처와 원문 이미지/PDF를 찾고 OCR 후보로 넘긴다.

## 다음 작업

3. 이미 확보된 최치원 32편, 정지상 15편, 허난설헌 파일럿 작품을 새 catalog에 매핑한다.
4. 홈페이지 공개용 작가 bio는 이 문서의 내용을 바탕으로 600~900자 자체 문장으로 다시 쓴다.
5. 번역은 기존 번역 권리가 확실한 경우만 가져오고, 그 외는 직접 번역 queue로 보낸다.

## 우선 확인해야 할 위험

- 고대 인물 중 상당수는 한문 정형시가 아니라 향가·가요·설화 속 시문 중심이다. 수집 대상에서 빼지 말고 `genre=form-native` 계열로 분리해야 한다.
- 기생·여성 문인 작품은 귀속 문제가 있을 수 있으므로 원문별 출처 확인이 중요하다.
- 근대 전환기 인물은 사망 연도와 저작권 상태를 반드시 확인해야 한다.
- 한국고전종합DB 번역은 참고용으로만 쓰고, 공개 번역은 직접 번역 또는 라이선스 확인 번역으로 제한한다.
