## 작업 두 갈래

### 갈래 A. 미분류 당대 재분류 (2,628명)

기존 `era_period='당'` 시인을 생몰 연도 기준 4분화에 자동 배정.

| 슬러그 | 생년 범위 |
|---|---|
| 초당 | 618 ≤ y < 712 (또는 사망연도 기준) |
| 성당 | 712 ≤ y < 766 |
| 중당 | 766 ≤ y < 835 |
| 만당 | 836 ≤ y ≤ 907 |

전제: `poets.life_birth`/`life_death` 신뢰성. 결측 시 보류 풀(`uncertain-tang`).

산출: `UPDATE poets SET era_period=...` SQL 마이그레이션 + 검증 리포트.

### 갈래 B. 공백 6시대 데이터 수집

각 시대별 PDF/위키문헌/한문 디지털 아카이브 → jds 파이프라인:

| 시대 | 1차 소스 후보 |
|---|---|
| 兩漢 | 古詩源, 古樂府 |
| 위진남북조 | 文選, 玉臺新詠, 古詩源 |
| 송 | 全宋詩 (北京大 데이터베이스), 宋詩鈔 |
| 원 | 元詩選 |
| 명 | 明詩綜 |
| 청 | 清詩匯, 晚晴簃詩匯 |

**참조 파이프라인**: `~/Documents/development/jds/` (전당시 PDF→파싱→AI 번역→DB 적재)

각 시대 1차 목표: 시인 30~50명 / 시 200~500수 (페이지 큐레이션 충분 분량).

## 우선순위 권장
