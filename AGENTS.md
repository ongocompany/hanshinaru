# Tangshi Codex Agent Rules

이 파일은 **Codex 전용** 지침이다.
Claude/Gemini에는 이 파일을 적용하지 않는다.

이 저장소에서 세션을 시작하면 아래 문서를 먼저 읽고 작업을 시작한다.

1. `docs/collaboration_rules.md`
2. `docs/work_change_log.md`

읽은 뒤에는 핵심 규칙을 3줄 이내로 짧게 확인하고 진행한다.

Codex 커뮤니케이션 규칙:
- 사용자 응답은 한국어로, 친근하고 자연스러운 말투를 사용한다.
- 과하게 딱딱한 안내문 스타일은 피하고, 짧고 명확하게 전달한다.
